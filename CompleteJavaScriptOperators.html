All JavaScript Operators

Complete reference of all JavaScript operators with examples and usage patterns.

Arithmetic Operators

Perform mathematical operations on numbers and variables.

+
Addition
Adds two numbers or concatenates strings
Examples:
5 + 3 // 8
'Hello' + 'World' // 'HelloWorld'
-
Subtraction
Subtracts the right operand from the left
Examples:
10 - 3 // 7
5 - 8 // -3
*
Multiplication
Multiplies two numbers
Examples:
5 * 3 // 15
2.5 * 4 // 10.0
/
Division
Divides the left operand by the right
Examples:
10 / 2 // 5
7 / 2 // 3.5
%
Modulus
Returns the division remainder
Examples:
10 % 3 // 1
15 % 4 // 3
**
Exponentiation
Raises the left operand to the power of the right
Examples:
2 ** 3 // 8
5 ** 2 // 25
++
Increment
Increases a number by 1
Examples:
let x = 5;
x++; // x = 6
++x; // x = 7
--
Decrement
Decreases a number by 1
Examples:
let y = 8;
y--; // y = 7
--y; // y = 6

Assignment Operators

Assign values to variables and perform operations simultaneously.

=
Assignment
Assigns a value to a variable
Examples:
let x = 10;
const name = 'John';
+=
Add & Assign
Adds and assigns in one operation
Examples:
let x = 5;
x += 3; // x = 8
-=
Subtract & Assign
Subtracts and assigns in one operation
Examples:
let x = 10;
x -= 4; // x = 6
*=
Multiply & Assign
Multiplies and assigns in one operation
Examples:
let x = 5;
x *= 3; // x = 15
/=
Divide & Assign
Divides and assigns in one operation
Examples:
let x = 20;
x /= 4; // x = 5
%=
Modulus & Assign
Calculates modulus and assigns
Examples:
let x = 17;
x %= 5; // x = 2
??=
Nullish Assignment
Assigns only if variable is null/undefined
Examples:
let x = null;
x ??= 10; // x = 10
x ??= 20; // x = 10

Comparison Operators

Compare values and return boolean results (true/false).

==
Equal To
Checks equality with type conversion
Examples:
5 == '5' // true
0 == false // true
‚ö†Ô∏è Avoid: Performs type coercion
===
Strict Equal
Checks equality without type conversion
Examples:
5 === '5' // false
5 === 5 // true
‚úÖ Recommended: No type coercion
!=
Not Equal
Checks inequality with type conversion
Examples:
5 != '5' // false
0 != false // false
!==
Strict Not Equal
Checks inequality without type conversion
Examples:
5 !== '5' // true
5 !== 6 // true
>
Greater Than
Checks if left is greater than right
Examples:
10 > 5 // true
3 > 3 // false
<
Less Than
Checks if left is less than right
Examples:
5 < 10 // true
5 < 5 // false
>=
Greater Than or Equal
Checks if left is greater or equal to right
Examples:
5 >= 5 // true
4 >= 5 // false
<=
Less Than or Equal
Checks if left is less or equal to right
Examples:
5 <= 5 // true
6 <= 5 // false

Logical Operators

Used with boolean values to combine or invert conditions.

&&
Logical AND
Returns true if both operands are truthy
Examples:
true && false // false
5 && 'hello' // 'hello'
||
Logical OR
Returns true if at least one operand is truthy
Examples:
false || true // true
0 || 'default' // 'default'
!
Logical NOT
Inverts the boolean value
Examples:
!true // false
!0 // true
??
Nullish Coalescing
Returns right operand if left is null or undefined
Examples:
null ?? 'hi' // 'hi'
0 ?? 'hi' // 0
‚úÖ Safer than || for 0, '', false

Bitwise Operators

Perform operations on the binary representation of numbers.

&
Bitwise AND
Performs AND on each pair of bits
Examples:
5 & 3 // 1 (101 & 011 = 001)
|
Bitwise OR
Performs OR on each pair of bits
Examples:
5 | 3 // 7 (101 | 011 = 111)
^
Bitwise XOR
Performs XOR on each pair of bits
Examples:
5 ^ 3 // 6 (101 ^ 011 = 110)
~
Bitwise NOT
Inverts all bits
Examples:
~5 // -6 (inverts 00000101 ‚Üí 11111010)
<<
Left Shift
Shifts bits left, fills with zeros
Examples:
5 << 1 // 10 (101 ‚Üí 1010)
>>
Sign-Propagating Right Shift
Shifts bits right, copies sign bit
Examples:
-8 >> 1 // -4
>>>
Zero-Fill Right Shift
Shifts bits right, fills with zeros
Examples:
-8 >>> 1 // 2147483644
